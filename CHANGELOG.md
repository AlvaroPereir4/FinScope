# Changelog

  - Feature: Readme and changelog added.
  - Feature: User login, registration, and session management using Flask-Login.
  - Feature: Overview of total income, expenses, balance, and net worth.
  - Feature: Income management (Add/Edit/Delete).
  - Feature: Expense management (Micro and Macro views).
  - Feature: Support for credit card installments logic.
  - Feature: Management of real account balances (Debit/Checking).
  - Feature: Card management (Limit, Closing Day, Due Day).
  - Feature: Invoice view generation based on closing dates.
  - Feature: Portfolio tracking (Fixed Income, Variable, Crypto).
  - Feature: Transaction history (Contributions, Earnings, Withdrawals).
  - Feature: Progress tracking for financial goals (Saving/Spending limits).
  - Feature: Customizable categories and buyers list.
  - Feature: Dark mode interface with responsive charts using Chart.js.
